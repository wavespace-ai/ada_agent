# Linear Solver Skill

## Instructions
To solve a linear equation of the form ax + b = 0, use the `solve.py` script.

### Usage
`python skills/math/solver/solve.py <a> <b>`

## Examples
User: "Solve 2x + 4 = 0"
Command: `python skills/math/solver/solve.py 2 4`
